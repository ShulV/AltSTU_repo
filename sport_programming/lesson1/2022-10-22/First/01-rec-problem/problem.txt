
Алгоритм вычисления функции F(n), где n - натуральное число, 
задан следующими соотношениями:

 F(n) = n, при n < 2,
 F(n) = F(n / 2) + 1, когда n >= 2 и чётное,
 F(n) = F(3n + 1) + 1 , когда n >= 2 и нечётное.

Найти количество значений n на отрезке [1;Х], 
для которых F(n) определено и больше 100.

input
Задано Х,  10 <=X <= 500

Output
Ответ на задачу


Пример

input
100

Output  
15
