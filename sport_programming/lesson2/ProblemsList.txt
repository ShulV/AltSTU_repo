    f(x) = x

03-for  borg
По каютам корабля передвигаются привидение и капитан.
Какова вероятность им НЕ встретиться в одной каюте или оказаться в соседних, когда они перемещаются навстречу друг другу?
Известен маршрут капитана.
Привидение бродит случайным образом.

РЕШЕНИЕ
Начальная вероятность привидения оказаться в каюте номерk
p(k) = <число ребер в k> / <всего ребер в графе>
Далее в цикле по длине маршрута пересчитываем верояиность привидения

06-DP   Temple Build
********************
Дана равностороняя пирамида. Есть кирпичи трех типов
Замостить ее квадратиками (кубиками) из числа заданных так,
чтобы (см картинку)
   - были ряды квадратов одного размера,
   - был максимальный объем

РЕШЕНИЕ
f(0) = 0  // f(k) - объем до  k-ой высоты включительно
f(k) = max( f(i) + volume(j))  // считаем по всем трем типам кирпичей
